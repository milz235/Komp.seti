Глава 1: Введение в HTML

HTML (HyperText Markup Language) — это язык разметки, используемый для создания веб-страниц. Он определяет структуру и содержание веб-документа с помощью тегов и атрибутов.

Основные элементы HTML:

  • <html> — корневой элемент документа.
  • <head> — содержит метаинформацию о странице.
  • <body> — основное содержимое страницы.

Пример простого HTML-документа:

<!DOCTYPE html>
<html>
<head>
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая веб-страница.</p>
</body>
</html>
