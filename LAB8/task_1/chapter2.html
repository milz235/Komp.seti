Глава 2: Основные теги HTML

В этой главе мы рассмотрим основные теги HTML, которые используются для создания структуры веб-страницы.

Основные теги:

  • <h1> - <h6> — заголовки разного уровня.
  • <p> — параграф текста.
  • <a> — ссылка.
  • <img> — изображение.
  • <ul>, <ol>, <li> — списки.

Пример использования тегов:

<h1>Заголовок 1 уровня</h1>
<p>Это параграф текста.</p>
<a href="https://example.com">Ссылка на сайт</a>
<img src="image.jpg" alt="Описание изображения">
<ul>
    <li>Пункт 1</li>
    <li>Пункт 2</li>
</ul>
